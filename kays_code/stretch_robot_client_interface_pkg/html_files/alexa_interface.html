How do communicate with your Stretch Robot with Alexa?

Always make sure you say Alexa

Then say either...

  • have Stretch Robot ...
  • tell Stretch Robot to ...
  • ask Stretch Robot to ...
  • make Stretch Robot ...

Then use one of the commands shown below to begin commanding your Stretch Robot!

Command Command's Function
Drive Forward This will have Stretch Robot drive forward a little
Drive Back This will have Stretch Robot drive backwards a little
Rotate Left This will have Stretch Robot drive backwards a little
Rotate Right This will have Stretch Robot drive backwards a little
Command Function
Stow Arm Will move the arm from its current position into the stow position for easier storage and mobility
Come Find Me Will activate an action of the Robot to search the area and locate a person
